Miller Indices

Definition: Any reciprocal vector $\vec{G}$ in the reciprocal lattice give multiples of $2\pi$ when dotted with all real vectors $\vec{R}$ in the real lattice. That is, if $\vec{G}$ is a reciprocal vector then: $$\vec{G}\cdot\vec{R} = 2\pi m \phantom{x} \text{for all } \vec{R} \text{ in real lattice}$$ Pick any $\vec{G}$ in the reciprocal lattice. For this $\vec{G}$, consider the set of all planes that have the form $\vec{G}\cdot\vec{r} = 2\pi m$. By definition of $\vec{G} \in \text{RL}$, it must be true that for all $\vec{R} \in \text{DL}$ that $\vec{R}$ is in one of the $\vec{G}\cdot\vec{r} = 2\pi m$ planes. However, the converse is in general not true: not all of the $\vec{G}\cdot\vec{r} = 2\pi m$ planes have to contain a point in $\text{RL}$. Anyways, what we have established so far is that to every reciprocal vector $\vec{G} \in \text{RL}$ there corresponds a set (not family) of planes in $\text{DL}$ that contain all the real lattice points, and the normal to these set of planes point in the same direction as $\vec{G}$. It also follows from the equation of the planes that the distance between the planes in the $\text{DL}$ is $d = 2\pi/|\vec{G}|$.

How do we maximize $d$? To maximize $d$, we pick the shortest possible $\vec{G}$ in that direction. Because we are still picking a valid reciprocal vector, our above correspondence still holds - therefore, our set of lattice planes in $\text{DL}$ will still contain all the lattice vectors $\vec{R}$. Further, because $d$ is maximized, there is intuitively no redundancy left in the lattice planes - all lattice planes will contain points in $\text{DL}$. (Please check this intuition)

Conclusion: If we pick the shortest possible reciprocal lattice vector $\vec{G}_\text{min}$ pointing in a given direction, then the set of planes $\vec{G}\cdot\vec{r} = 2\pi m$ will correspond to a family of lattice planes which normal points in the direction of $\vec{G}_\text{min}$.

The $\text{DL}$ of a $\text{fcc}$ lattice is: $$\vec{a}_1 = \frac{a}{2}(0, 1, 1), \phantom{x} \vec{a}_2 = \frac{a}{2}(1, 0, 1), \phantom{x} \vec{a}_3 = \frac{a}{2}(1, 1, 0)$$ The corresponding $\text{RL}$ is (recall that the $\text{RL}$ of $\text{bcc}$ is $\text{fcc}$ and $\text{RL}$ of $\text{fcc}$ is $\text{bcc}$) : $$\vec{b}_1 = \frac{2\pi}{a}(-1, 1, 1), \phantom{x} \vec{b}_2 = \frac{2\pi}{a}(1, -1, 1), \phantom{x} \vec{b}_3 = \frac{2\pi}{a}(1, 1, -1)$$

How do we define the Miller indices for $\text{fcc}$? The idea is this. Even though we are interested in $\text{fcc}$, we are going to start with the $\text{DL}$ of a simple cubic ($\text{c}$) lattice with side length $a$: $$\vec{a}_1 = \frac{a}{2}(1, 0, 0), \phantom{x} \vec{a}_2 = \frac{a}{2}(0, 1, 0), \phantom{x} \vec{a}_3 = \frac{a}{2}(0, 0, 1)$$ The $\text{RL}$ is: $$\vec{b}_1 = \frac{2\pi}{a}(1, 0, 0), \phantom{x} \vec{b}_2 = \frac{2\pi}{a}(0, 1, 0), \phantom{x} \vec{b}_3 = \frac{2\pi}{a}(0, 0, 1)$$ Now, we make the following important observation: the $\text{RL}$ of $\text{c}$ is a superlattice of the $\text{RL}$ of $\text{fcc}$. In other words, all the lattice points in the $\text{RL}$ of $\text{fcc}$ are also lattice points in the $\text{RL}$ of $\text{c}$, but not vice versa.

Therefore, we can describe the lattice points in the $\text{RL}$ of $\text{fcc}$ by enforcing constraints on the Miller indices, which are still defined with respect to the $\text{RL}$ of $\text{c}$.

For $\text{fcc}$, the constraint is that $(h,k,l)$ are all even or $(h,k,l)$ are all odd. This can be proven by writing linear combinations of the true $\vec{b}_1, \vec{b}_2, \vec{b}_3$ vectors: $$ \begin{align} \vec{G_{\text{fcc}}} &= H\vec{b}_1 + K\vec{b}_2 + L\vec{b}_3 \\ &= \frac{2\pi}{a}(-H+K+L, H-K+L, H+K-L) \end{align}$$ In this case $H$, $K$ and $L$ (the "Miller indices" for the true reciprocal lattice of $\text{fcc}$) can take any values. Now, notice that regardless whether $(H,K,L)$ is $(e, e, e)$, $(e, e, o)$, $(e, o, o)$, $(o, o, o)$, $(h,k,l)$ must either all be odd or all be even.

We can repeat a similar analysis with $\text{bcc}$. For convenience, the $\text{DL}$ is: $$\vec{a}_1 = \frac{a}{2}(-1, 1, 1), \phantom{x} \vec{a}_2 = \frac{a}{2}(1, -1, 1), \phantom{x} \vec{a}_3 = \frac{a}{2}(1, 1, -1)$$ and the $\text{RL}$ is: $$\vec{b}_1 = \frac{2\pi}{a}(0, 1, 1), \phantom{x} \vec{b}_2 = \frac{2\pi}{a}(1, 0, 1), \phantom{x} \vec{b}_3 = \frac{2\pi}{a}(1, 1, 0)$$ The consequence for the Miller indices $(h,k,l)$ is that $h+k+l$ is even.
